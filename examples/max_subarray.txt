Write a function called find_max_subarray_sum that finds the maximum sum of a contiguous subarray.

Function Name: find_max_subarray_sum
Parameter: nums (List[int])
Return: int (maximum subarray sum)

Description:
Given an integer array, find the contiguous subarray (containing at least one number) which has the largest sum and return its sum.
This is known as Kadane's Algorithm problem.

Examples:
- find_max_subarray_sum([1, 2, 3, 4]) = 10
- find_max_subarray_sum([-2, 1, -3, 4, -1, 2, 1, -5, 4]) = 6 (subarray [4, -1, 2, 1])
- find_max_subarray_sum([-1, -2, -3]) = -1
- find_max_subarray_sum([5]) = 5

Edge Cases:
- Single element array
- All negative numbers
- All positive numbers
- Empty array (return 0)
- Mix of positive and negative numbers

Constraints:
- Array may contain negative numbers
- Array may be empty (return 0)
- At least one element must be included in the subarray
