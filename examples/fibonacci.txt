Write a function called fibonacci that returns the nth Fibonacci number.

Function Name: fibonacci
Parameter: n (integer, n >= 0)
Return: integer (nth Fibonacci number)

Description:
The Fibonacci sequence is defined as:
- F(0) = 0
- F(1) = 1
- F(n) = F(n-1) + F(n-2) for n > 1

Examples:
- fibonacci(0) = 0
- fibonacci(1) = 1
- fibonacci(5) = 5
- fibonacci(10) = 55

Edge Cases:
- Handle n = 0
- Handle n = 1
- Optimize for larger values

Constraints:
- n >= 0
- n <= 30
